# مرجع الأيقونات في تطبيق نبض الملاعب

هذا الملف يحتوي على قائمة شاملة بجميع الأيقونات المستخدمة في التطبيق، وهو بمثابة دليل لك لتحديد موقعها، فهم كيفية استخدامها، وتغييرها إذا لزم الأمر.

## 1. أيقونات مخصصة (Custom SVG Icons)

هذه الأيقونات هي مكونات React تحتوي على كود SVG مباشر. يمكنك تعديل تصميمها مباشرة عن طريق تعديل ملفها.

---

### شعار التطبيق الرئيسي
- **المسار:** `src/components/icons/NabdAlMalaebLogo.tsx`
- **الاستخدام:** يظهر في الشريط العلوي، شاشات التحميل، وشاشة البداية.
- **الحجم النموذجي:**
    - الشريط العلوي: `h-6 w-6` (24px) - يتم التحكم به في `src/components/ScreenHeader.tsx`.
    - شاشة البداية: `h-24 w-24` (96px) - يتم التحكم به في `src/app/screens/WelcomeScreen.tsx`.

## 2. أيقونات مكتبة `lucide-react`

هذه الأيقونات لا تملك ملفات منفصلة في المشروع، بل يتم استيرادها كـ "مكونات" من مكتبة `lucide-react`. يتم التحكم في حجمها ولونها مباشرة عند استخدامها في الكود عبر فئات CSS (مثل `h-5 w-5`, `text-primary`).

فيما يلي قائمة بأبرز هذه الأيقونات، أماكن استخدامها، وأحجامها النموذجية:

- **`ArrowLeft` (سهم لليسار):**
    - **الاستخدام:** زر الرجوع في الشريط العلوي.
    - **المكان:** `src/components/ScreenHeader.tsx`
    - **الحجم:** `h-5 w-5` (20px)

- **`Search` (بحث):**
    - **الاستخدام:** زر البحث في الأشرطة العلوية.
    - **المكان:** `src/app/screens/MatchesScreen.tsx`, `src/app/screens/NewsScreen.tsx`, `etc.`
    - **الحجم:** `h-5 w-5` (20px)

- **`Star` (نجمة):**
    - **الاستخدام:** لتحديد الفرق والبطولات المفضلة.
    - **المكان:** `src/app/screens/CompetitionsScreen.tsx`, `src/components/SearchSheet.tsx`, `etc.`
    - **الحجم:** `h-5 w-5` (20px)

- **`MoreHorizontal` (المزيد):**
    - **الاستخدام:** أيقونة تبويب "المزيد" في الشريط السفلي.
    - **المكان:** `src/components/BottomNav.tsx`
    - **الحجم:** `h-6 w-6` (24px)

- **`Settings` (إعدادات):**
    - **الاستخدام:** أيقونة "الإعدادات العامة" في شاشة المزيد.
    - **المكان:** `src/app/screens/SettingsScreen.tsx`
    - **الحجم:** `h-6 w-6` (24px)

- **`Pencil` (قلم رصاص):**
    - **الاستخدام:** للتعديل (مثل تعديل الأسماء المخصصة).
    - **المكان:** `src/app/screens/AllCompetitionsScreen.tsx`, `src/app/screens/TeamDetailScreen.tsx`
    - **الحجم:** `h-4 w-4` (16px)

- **`Square` (مربع):**
    - **الاستخدام:** لتمثيل البطاقات الصفراء والحمراء في تفاصيل المباراة.
    - **المكان:** `src/app/screens/MatchDetailScreen.tsx`
    - **الحجم:** `h-5 w-5` (20px)

- **`ArrowUp` / `ArrowDown` (أسهم التبديل):**
    - **الاستخدام:** لتمثيل تبديلات اللاعبين في تفاصيل المباراة.
    - **المكان:** `src/app/screens/MatchDetailScreen.tsx`
    - **الحجم:** `h-3 w-3` (12px)

- **`Loader2` (أيقونة التحميل):**
    - **الاستخدام:** تظهر عند تحميل البيانات.
    - **المكان:** في معظم شاشات التطبيق.
    - **الحجم:** يختلف حسب السياق (`h-8 w-8`, `h-4 w-4`, `etc.`).

- **`X` (إغلاق):**
    - **الاستخدام:** لإغلاق النوافذ المنبثقة.
    - **المكان:** `src/components/ui/dialog.tsx`, `src/components/ui/sheet.tsx`
    - **الحجم:** `h-4 w-4` (16px)
